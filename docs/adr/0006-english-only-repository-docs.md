# ADR 0006 — English-Only Repository Docs

Date: 2026-06-13
Status: Accepted

## Context

NeNe Payout targets an international developer audience and follows the NeNe ecosystem pattern.

## Decision

All files in `docs/`, `README.md`, `AGENTS.md`, `CLAUDE.md`, ADRs, OpenAPI, and code identifiers are written in English. Japanese is allowed in GitHub Issues, PR bodies, commit descriptions, and `.cursor/rules/` (team workflow notes).

## Consequences

- Consistent with sibling products (nene-vault, nene-field, etc.)
- Wider developer accessibility
